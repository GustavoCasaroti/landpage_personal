# 💻 Gustavo Casaroti - Portfolio

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, consumindo dados da API do GitHub em tempo real.

## 🚀 Funcionalidades
- Integração com API do GitHub
- Exibição de estatísticas (stars, repos, followers)
- Linguagens mais utilizadas
- Lista de repositórios recentes
- Animações e efeitos interativos
- Favicon dinâmico animado

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub API
